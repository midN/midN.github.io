2016 Reading list

Author: Alexander Dymo

If you have ever came into some piece of code in Application where you just had no idea how to make it faster and better, then read this book. This book really shows you what software to use in order to optimize something efficiently and how to find slowest parts of your code.

Author: Sandi Metz

This is the best book that explains in easy-language how Object-Oriented Design works. If you ever came into an old project that is pure hell to update and manage, and you don't want to end to up with your project same way in couple of years, then definitely read this book.
